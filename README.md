# Intra-hippocampal Connectivity in the Psychosis Risk State
This repository contains code associated with a project investigating intra-hippocampal connectivity in individuals at clinical high risk for psychosis (CHR-P). A preprint describing this work is available at https://doi.org/10.64898/2026.03.10.26348090.

The project uses demographic, clinical, and neuroimaging data from the North American Prodromal Longitudinal Study 3 (NAPLS-3). Data access is governed by the NIMH Data Archive and requires application through a principal investigator affiliated with an approved research institution. Consequently, the underlying data cannot be shared publicly. This repository therefore provides only the analysis code used in the project.
